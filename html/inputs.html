Input Components

Input blocks use a series of classes which stay the same at the basis, the only main difference is the animation name.

Example:

<span class="input input--ANIMATION">
<input type="text" class="input__field">
<label class="input__label EXTRACLASSES">
<span class="input__label-content content">
Username
</span>
</label>
</span>

Underline Animated

A simple underline animation with multi color support

Uses class input--underline

Color classes input__label-color[1-3]

Lines Animated

Input with top & bottom borders that move for text

Uses class input--lines

Move Top Animated

Input with top & bottom borders that move for text

Uses class input--move-top

Requires a data-content attribute to display the top label

Box Bottom Animated

Animates a label to the bottom of the box and highlights it

Uses class input--box-bottom

Split Animated

Animates Split input with label moving to bottom

Uses class input--split

Flash Animated

Animates Flash border on click of a input

Uses class input--flash
