Button Components

Responsive: Yes

Button blocks use simple css classes to set their animation

Attribute Classes:

  • cb--round

Example:


<button class="cb cb__{ANIMATION}">Button</button>
<button class="cb cb__{ANIMATION} cb--round">Button</button>
