Button Components

Responsive: Yes

Button blocks use simple css classes to set their animation

Example:

<button class="cb cb-{ANIMATION}">Button</button>

Corners Animation

Expands to opposite corners from center animation

Uses class cb-corners the 2nd button is using the cb-round helper

Swipe Right Animation

Swipe right button fill animation

Uses class cb-rswipe the 2nd button is using the cb-round helper

Swipe Down Animation

Swipe down button fill animation

Uses class cb-dswipe the 2nd button is using the cb-round helper
