Creative Sass

Sass Components is a little library of SASS/CSS driven animations for all kinds of elements, it is ever growing and more will continue to be added.

Creative Sass Library uses the Creative Commons License, feel free to use these components as much as you'd like.

Color Variables

There are a series of Sass variables used for colors in Sass Components, such as:

  • $primary - Used for primary color points
  • $primary-accent - Used for secondary coloring
  • $secondary-accent - Used for offset colors
  • $syntax - Used for these code highlights and other coloring
  • $shade-[100, 150, 200, 900] - Used for darker shades around elements

Specific variables for coloring in each animation can be seen in the variables.scss before compiling.

Utilities

There are several built in utils used in Sass Components

  • .img-circle - Sets the border radius to a circle
  • .round - like .img-circle sets the element to be rounded, but also sets it to a specific size and removes padding
  • .hidden - Mainly for bootstrap 4 users, since the hidden util is gone
  • .highlight - Uses shade-150 and $syntax to create the text highlight
