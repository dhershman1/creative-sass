Ripple Component

Responsive: Yes

Creates a Ripple animation over the desired element (the cards below are also rippled if you click them)

Example:


<button class="ripple HELPERS">Button Ripple!</button>
<td class="ripple HELPERS">Table Cell Ripple!</td>
<tr class="ripple HELPERS">Table Row Ripple!</tr>
<table class="ripple HELPERS">Table Ripple!</table>

Helper Classes

  • dark - Creates a Dark Ripple Color
  • light - Creates a Light Ripple Color

Regular Buttons

Ripple over dark/light colored buttons

Uses class .ripple.dark and .ripple.light

Colored Buttons

Ripple over colored buttons

Uses class .ripple.dark and .ripple.light

Tables

Ripple over tables, table rows, and table cells

Ripple cell! Ripple cell!
Ripple row! Ripple row!
Even the table! Has ripples!

Uses class .ripple.dark and .ripple.light
