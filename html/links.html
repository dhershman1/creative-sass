Link Components

Responsive: Semi

Link animations for anchor tags, apply the animation to each anchor tag after applying the block tag to its wrapper

Variables:

  • $link-font-color: Main font color for normal links
  • $link-light-color: Main font color for links with the dark bg
  • $link-main-bg: Main color for transition links

Example:


<div class="cl">
<a class="cl--{ANIMATION}" href="#">Link 1</a>
<a class="cl--{ANIMATION}" href="#">Link 2</a>
<a class="cl--{ANIMATION}" href="#">Link n..</a>
</div>
