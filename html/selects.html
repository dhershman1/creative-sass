Select Components

Responsive: Yes

Select blocks use simple css classes to set their animation. You need to create a list to mock out your select box as well as a placeholder

Example:


<div class="cs-select cs-slide" tabindex="0">
<span class="cs-placeholder">PLACEHOLDER</span>
<div class="cs-options">
	<ul>
		<li class="CLASSES" data-option="" data-value="VALUE"><span>TEXT</span></li>
	</ul>
</div>
<select class="cs-select">
	<option value="" disabled selected>PLACEHOLDER</option>
	<option value="VALUE">TEXT</option>
</select>
</div>
