Select Components

Select blocks use simple css classes to set their animation

Example:


  <select class="cs-select cs-{ANIMATION}">
    <option value="text">Text</option>
  </select>

Slide Animation

Slides in Menu. Uses icomoon for the icon library

Uses class cs-slide

To set icons use data-class="icon-opt[1-5]"

Make sure you set those in the sass to your desired icons content!

Underline Animation

Slides in underline style menu. Uses icomoon for the icon library

Uses class cs-underline

Circular Animation

Creates a cirlce grid menu.

Uses class cs-circle
