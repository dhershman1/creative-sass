Modal Components

Modals consist of html placed on the page with a md-overlay class on a div element This produces both the modal and the screen overlay for the page. Modal must have an id attribute with the modal name

Classes

  • md-modal - Says a div will be used as a modal
  • md-{ANIMATION} - Determine the Modal animation
  • md-content - Sets the content wrapper use an h3 tag in this wrapper for a header
  • md-close - Closes the modal
  • md-overlay - Will open an overlay behind the modal
  • md-trigger - Will fire the modal when clicked set data-modal attribute in the same element with the modal name 

				<div class="md-modal md-fade-slide" id="modal-test">
				 <div class="md-content">
				  <h3>Test Modal</h3>
					 <p>Modal Content</p>
					 <button class="btn btn-rswipe">Modal Button</button>
					 <button class="md-close btn btn-rswipe">Close</button>
				 </div>
				</div>
				<button class="md-trigger" data-modal="modal-test">Modal Trigger</button>
				<div class="md-overlay"></div>
