Modal Components

Responsive: Semi

Modals consist of html placed on the page with a cm-overlay class on a div element This produces both the modal and the screen overlay for the page. Modal must have an id attribute with the modal name

Classes

  • cm-modal - Says a div will be used as a modal
  • cm-{ANIMATION} - Determine the Modal animation
  • cm-content - Sets the content wrapper use an h3 tag in this wrapper for a header
  • cm-close - Closes the modal
  • cm-overlay - Will open an overlay behind the modal
  • cm-trigger - Will fire the modal when clicked set data-modal attribute in the same element with the modal name 

<div class="cm-modal cm-effect-{{ANIMATION}}" id="modal-test">
 <div class="cm-content">
  <h3>Test Modal</h3>
   <p>Modal Content</p>
   <button class="cb cb-rswipe">Modal Button</button>
   <button class="cm-close cb cb-rswipe">Close</button>
 </div>
</div>
<button class="cm-trigger" data-modal="modal-test">Modal Trigger</button>
<div class="cm-overlay"></div>
