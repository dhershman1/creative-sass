Modal Components

Responsive: Yes

Modals consist of html placed on the page with a cm-overlay class on a div element This produces both the modal and the screen overlay for the page. Modal must have an id attribute with the modal name

Classes

  • cm__modal - Says a div will be used as a modal
  • cm--{ANIMATION} - Determine the Modal animation
  • cm__content - Sets the content wrapper use an h3 tag in this wrapper for a header
  • cm__close - Closes the modal
  • cm__overlay - Will open an overlay behind the modal
  • cm__trigger - Will fire the modal when clicked set data-modal attribute in the same element with the modal name 

<div class="cm__modal cm--{ANIMATION}" id="modal-test">
<div class="cm__content">
<h3 class="cm__title">Test Modal</h3>
 <p class="cm__text">Modal Content</p>
 <button >Modal Button</button>
 <button class="cm__close">Close</button>
</div>
</div>
<button class="cm__trigger" data-modal="modal-test">Modal Trigger</button>
<div class="cm__overlay"></div>
