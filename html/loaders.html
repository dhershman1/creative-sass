Loader Component

Responsive: Yes

Create a responsive loading animation

Uses $loader-height & $loader-width variables

Examples:


<div class="cs-loader">
  <div class="cs-loader__squares cs-loader__squares--square-1"></div>
  <div class="cs-loader__squares cs-loader__squares--square-2"></div>
  <div class="cs-loader__squares cs-loader__squares--square-3"></div>
  <div class="cs-loader__squares cs-loader__squares--square-4"></div>
</div>

<div class="cs-loader">
  <svg class="circles">
    <circle class="circles__internal" cx="60" cy="60" r="30"></circle>
    <circle class="circles__external" cx="60" cy="60" r="50"></circle>
  </svg>
</div>

Squares

Square based rotating loading animation

Setup Class: .cs-loader__squares

Attribute Class: .cs-loader__squares--square-[1-4]

Variables:

  • $loader-squares-h: Height value for each square
  • $loader-squares-w: Width value for each square
  • $square-[1-4]-bg: Background color of the corrisponding square

Circle

Circle loader animation

Div Wrapper Class: .cs-loader

Wrapper Class: .cs-loader__circles

Internal Class: .cs-loader__circles--internal

External Class: .cs-loader__circles--external

Variables:

  • $circle-stroke: The color stroke the circles base off of
