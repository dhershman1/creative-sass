Loader Component

Responsive: Yes

Create a responsive loading animation

Examples:


<div class="cs-loader">
  <div class="squares square-1"></div>
  <div class="squares square-2"></div>
  <div class="squares square-3"></div>
  <div class="squares square-4"></div>
</div>

<div class="cs-loader">
  <svg class="circles">
    <circle class="internal-circle"> cx="60" cy="60" r="30"</circle>
    <circle class="external-circle"> cx="60" cy="60" r="50"</circle>
  </svg>
</div>

Squares

Square based rotating loading animation

Type Class: .squares

Piece Class: .square-1

Circle

Circle loader animation

Wrapper Class: .circles

Internal Class: .internal-circle

External Class: .external-circle
